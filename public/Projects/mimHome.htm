MIM Smart Home video

  • Abstract

  • The aim of the project is to simulate the smart home systems and include the idea of the remate control using a wesite " mimhome.ae.tn " to monitor the temperature and gas ratio, use a close loop control to get the temperature from web and control it by Arduino and finally the use of RFID technology for identification of the people in home.

  • Features

  • Learn the web programming and to build our own home site
  • Monitor the temperature and gas with the ability to add any physical quantity to display on site
  • A login page that is connected to database for multiple users enters and authorization
  • Connecting two controllers by enable and two data lines each for different user -card and medal- and connect the RFID system to the website
